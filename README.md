# Proyecto de Educación - Backend

Backend para el sistema de gestión educativa, desarrollado con **Node.js**, **Express** y **MySQL**.

## Instalación

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Configurar el archivo `.env` con las credenciales de la base de datos.

3. Iniciar el servidor:
   ```bash
   npm run dev
   ```
